# A.R.I.A. Translate

Desktop app for translating Online/Offline between **Italian**, **English** and **German**, with two interchangeable translation engines:

- **Local (Ollama)** — fully offline, no account or internet required, once the model is downloaded. Slower (roughly 15-60s per translation depending on the model).
- **Cloud (Groq)** — requires internet and a free [Groq](https://console.groq.com/keys) API key. Sub-second translations, at the cost of sending the text to Groq's servers.

The active engine is switched from the **Engine** menu ("Local (Ollama, offline)" / "Cloud (Groq, requires internet)") and shown live in the top-right status badge ("Local Translator Active" / "Internet Translator Active"). Every translation is generated in two registers: **formal** (high-level business/executive correspondence) and **informal** (casual, everyday tone).
