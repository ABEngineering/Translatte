# A.R.I.A. Translate

Offline desktop app for translating between **Italian**, **English** and **German**, powered by a local LLM agent running through **Ollama** (no internet connection required to translate, once the model is downloaded). Every translation is generated in two registers: **formal** and **informal**.

Creator: **Amaro Balsamo**
